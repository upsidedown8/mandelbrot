# mandelbrot
Visualiser for the Mandelbrot set, which allows the user to navigate the fractal.

![Mandelbrot GIF](./mandelbrot.gif)
![Mandelbrot PNG](./mandelbrot.png)

## Controls

| Action      | Shortcut    |
| ----------- | ----------- |
| Zoom out    |`-`          |
| Zoom in     |`=`          |
| -Iterations |`Q`          |
| +Iterations |`E`          |
| Up          |`W`          |
| Left        |`A`          |
| Down        |`S`          |
| Right       |`D`          |
